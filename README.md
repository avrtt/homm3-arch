This repository contains original source files for **Heroes of Might and Magic III: Complete** patched for Arch Linux. Includes both expansion packs (Armageddon's Blade & The Shadow of Death).

## Installation

1. Install [wine](https://archlinux.org/packages/extra/x86_64/wine/)
2. Clone the repo
3. Navigate to the folder 
4. Download `game_info.tar.gz` ([Drive]()) to the folder
5. Unpack media files:
```
tar -xvzf game_info.tar.gz
```
6. Run the script:
```
./start.sh
```
Wait for Wine to create the prefixes, configure the launcher and enjoy.
